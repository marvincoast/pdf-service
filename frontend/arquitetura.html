📥 1. Ingestão (PyMuPDF)

Leitura mecânica do PDF com extração de texto página por página e metadados (autor, título).

  • Alta performance (biblioteca C)
  • Extração por página com contagem de chars
  • Metadados do PDF (autor, título, datas)

🎯 2. Classificação

Algoritmo de scoring por keywords com 10 tipos de documento e 13+ keywords cada.

  • Currículo, Extrato, Receita, Certificado
  • NF, Contrato, Laudo, Jurídico, Técnico
  • Score de confiança percentual

🧠 3. Motor Cognitivo

LLM local via Ollama com prompts adaptativos que pedem resumo detalhado + entidades + recomendações.

  • Contexto: 4096 tokens
  • Output: até 800 tokens
  • Prompt específico por tipo de documento
  • Timeout: 90s (CPU)

🔄 4. Fallback Inteligente

Se a IA falhar, regex patterns extraem dados críticos: CPF, CNPJ, emails, telefones, valores monetários.

  • Handlers específicos por tipo
  • Extração de entidades nomeadas
  • Sempre garante resposta

⚡ 5. Backend Modular

Flask com Blueprint, app factory, e módulos separados para cada responsabilidade.

  • config.py → routes → services → utils
  • Gunicorn com 2 workers
  • Docker + docker-compose

🎨 6. Frontend Imersivo

Interface glassmorphism com HUD dinâmico, barra de progresso e exportação JSON.

  • CSS/JS modular e separado
  • 3D Parallax + Page Transitions
  • Copiar/Exportar JSON
